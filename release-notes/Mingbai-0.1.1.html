Mingbai 0.1.1

Fixed

  • Prevent hover lookups from triggering when the pointer is only near Chinese text.
  • Refresh permission-dependent hover state after local permission changes.
  • Restore hover lookups while still rejecting near-misses beside Chinese text.
  • Stop hover lookups from snapping to the first word of a line in apps (such as Messages) that report an unverifiable character position.
  • Fix hover lookup on large Chinese text embedded in images.
